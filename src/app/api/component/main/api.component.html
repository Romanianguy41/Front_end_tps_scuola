API

Student API All API that interact with the students
GET webapi/scuola/studente Get all students

This API request return all the students in the database

Parameters:

No parameters
GET webapi/scuola/studente/filtred Get filtred students

This API request return all the students in the database that match the search querry

Parameters:

Query params: search

Description

search: key value pair, using ' ' characters for string elements

Example:

search=nome:'francesco'

where nome is the key and 'francesco' is the value
POST webapi/scuola/studente Create new student

This API will add a new student to the DB

Body:

The data of the relative student

    }'}}

PUT webapi/scuola/studente/classe/{{'{idStudente}'}} Remove a student from a class

This API request will remove the student from the current class

Parameters:

Path params: idStudente

Description

To use this API you need to pass in the URL the studentID in this way you can remove his current class

Body:

No body
DELETE webapi/scuola/studente/{{'{idStudente}'}} Delete a student

This API request removes from the database the user that match the idStudente

Parameters:

Path params: idStudente

Description

To use this API you need to pass in the URL the studentID in this way you can delete the student

Body:

No body
